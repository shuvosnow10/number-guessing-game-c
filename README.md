# 🎯 Number Guessing Game in C

A simple console-based number guessing game written in C.
The computer randomly selects a number, and the player tries to guess it with hints provided after each attempt.

---

## 🚀 Features

* Random number generation using `rand()`
* Infinite loop until correct guess
* Hint system (higher/lower)
* Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

* C Programming Language
* Standard Libraries:

  * stdio.h
  * stdlib.h
  * time.h

---

## ▶️ How to Run

### 1. Compile the program

```bash
gcc main.c -o game
```

### 2. Run the program

```bash
./game
```

---

## 🎮 How to Play

* The computer chooses a number between **0–9**
* Enter your guess
* You will get hints:

  * "Go bigger"
  * "Go smaller"
* Keep guessing until you win 🎉

---

## 📌 Example Output

```
I have a number in my mind (0-9). Can you guess it?

Enter your Guess -> 5  
Wrong guess... TRY AGAIN (Go bigger!)

Enter your Guess -> 8  
You are right..! ;-)
```
